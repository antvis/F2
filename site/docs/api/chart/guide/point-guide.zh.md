---
title: 点标注 - PointGuide
---

## 用法

```jsx
import { PointGuide } from '@antv/f2';

<PointGuide records={[{ genre: 'Sports', sold: 5 }]} offsetX={10} offsetY={-10} style={{ fill: '#f00' }} />
```

## Props

- **records: Array**  
  标注的数据项或比例值
- **offsetX: number**  
  x 轴偏移量
- **offsetY: number**  
  y 轴偏移量
- **style**  
  标注样式
