# API

全局命名空间 `F2`，包含：

## 常量 Constants

```js
F2.version // F2 的版本
```

## 类 Classes

* [`F2.Chart`](./chart.md)：图表的入口类
* [F2.Shape](../develop/shape.md)：用于自定义 Shape 的入口
* [F2.G](../develop/graphic.md)：提供的绘图 API，替换原生的 canvas 接口，提供便利性
* [F2.Global](./global.md)：用于设置全局的样式和更改皮肤
* `F2.Util`：常见的工具方法
* [F2.Plugin](../develop/plugin.md): 用于获取插件
